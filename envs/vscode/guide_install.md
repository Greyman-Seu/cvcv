## 3.2 免密码登陆
简单版本
```bash
# 笔记本操作
ssh-keygen -t rsa
ssh-copy-id user@host_ip（远程ip）
# 案例
ssh-keygen -t rsa
ssh-copy-id yangkun.zhu@xxxip
# 测试
ssh yangkun.zhu@xxxip 不需要密码了，就是设置成功了
```
